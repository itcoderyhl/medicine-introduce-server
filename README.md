# medicine-introduce-server
基于SpringBoot+Vue前后端分离的银翘解毒液药品宣传网站，系统包含管理员、医生、用户角色，功能如下： 管理员：首页统计（帖子统计、七天咨询统计）；用户管理；医生管理；药品管理（银翘解毒液、其他药品）；知识文章；论坛交流（板块管理、帖子管理、帖子评论）；咨询管理；用药记录；系统留言；系统公告；个人信息；密码修改。 医生：知识文章；咨询管理；用药记录；公告信息；修改个人资料；修改密码。 用户：注册、登录系统；首页；产品介绍（同类产品对比）；专业知识；论坛交流（论坛版块、帖子列表、详情、点赞、收藏、评论）；专家咨询（咨询）；系统留言；公告信息。用户还具有个人中心功能，功能包括：我的帖子、帖子评论、帖子点赞、帖子收藏、我的咨询、用药历史、系统留言、个人资料、修改密码。
